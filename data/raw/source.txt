Data sources and provenance:
===========================================

1. PRIMARY SOURCES
------------------
- Synthetic examples created for the Gemma 4 Good project
- Inspired by real-world issues in environment, education, and healthcare
- Domain expert review by contributors from sustainability, pedagogy, and clinical backgrounds

2. AUGMENTATION SOURCES
-----------------------
- Paraphrase generation via back-translation (en→fr→en, en→de→en)
- Template-based expansion using domain-specific sentence frames
- Keyword substitution using approved domain vocabularies

3. CORPUS SOURCES
-----------------
- Aggregated from public reports (IPCC, UNESCO, WHO) summary statements
- Rewritten for neutrality and brevity
- No copyrighted text reproduced verbatim

4. PROMPT/RESPONSE SOURCES
--------------------------
- Prompts designed to cover a range of cognitive complexity (Bloom's taxonomy aligned)
- Responses initially generated by language model, then human-edited for accuracy
- Verified responses flagged and locked after domain expert review

5. VALIDATION SOURCES
---------------------
- Test set labels assigned by independent human annotators (not involved in training set creation)
- Inter-annotator agreement: Cohen's κ = 0.94 on test set

6. ETHICAL NOTES
----------------
- No personally identifiable information (PII) in any dataset
- No patient data, student records, or proprietary environmental monitoring data used
- All data is synthetic or derived from publicly available, non-sensitive summaries
- Bias auditing performed across label distributions; class balance within ±5%

7. VERSION HISTORY
------------------
- v1.0.0 (2025-01-15): Initial release — 6 train, 3 test, 3 prompts/responses
- v1.5.0 (2025-03-20): Expanded to 30 train, 10 test; added confidence scores
- v2.0.0 (2025-07-11): Full upgrade — 100 train, 30 test, 12 prompts/responses (simple→complex), 
  enriched metadata, augmented samples flagged, data dictionary v2, corpus expanded to 25 lines,
  multi-tier prompt complexity, quality-graded responses, domain source tracking in samples
